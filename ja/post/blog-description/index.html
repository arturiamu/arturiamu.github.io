前文

ずっと自分のブログを始めたいと思っていましたが、いつも忙しくて(怠惰で)、卒業が近づいた頃にようやく時間を見つけて気づこうと決めました。 このブログには、主に次の 2 つの目的があります：

人生と思いを記録する

ブログはあなたの経験、感情、考え、意見を言葉の形で記録することができ、それはあなたがあなたの考えをよりよく明確にし、経験を要約するのを助けるだけでなく、あなたの精神的な旅を他の人と共有するのにも役立ちます。

コミュニケーションと学習

ブログはコミュニケーションと学習のプラットフォームにもなり、自分の経験や洞察を他のブロガー、読者、仲間と交換して共有し、インスピレーションや学習の機会を増やすことができます。

このブログについて

このブログは「Hugo + Githubページ」を使用して構築されており、詳細については 如何用 GitHub Pages + Hugo 搭建个人博客 と 个人博客记 —— Github pages 绑定个人域名 を参照してください。 また、外国語学習を目的として、中国語版、英語版、日本語版のブログを同時に公開するように努め、言語や内容の誤りを見つけた場合は、批判や訂正を歓迎します。

このブログでは、セカンドレベルドメイン名 blog.mulinbiao.com を使用して arturiamu.github.io に解決し、blog.mulinbiao.comドメイン名には更新、ICPファイリング、その他の問題が含まれるため、接続を保証できないため、アクセスには arturiamu.github.io を使用することをお勧めします。
