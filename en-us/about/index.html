About the Hugo

Hugo is a static website generator written in Go language,using the Apache 2.0 open source license

Hugo supports configuration files in three formats:TOML/YAML/JSON

Hugo supports pages in Markdown and HTML formats,and also supports the use of shorts codes to enrich page content

Hugo natively supports multi-language,page classification,image processing,etc.

Hugo generates websites very fast and can be quickly deployed to private servers,commercial servers,GitHub Pages,Netlify,etc.

Official homepage gohugo.io

GitHub homepage https://github.com/gohugoio/hugo
